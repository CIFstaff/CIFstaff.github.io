Useful Links

Below are a collection of useful image processing programs, forums, and manuals

Software

FIJI (Fiji Is Just Image J)

Fiji is an image processing package — a "batteries-included" distribution of ImageJ, bundling many plugins which facilitate scientific image analysis.
Nikon Elements Image Viewer

NIS-Elements Viewer is a free standalone program to view image files and datasets. This version is mainly for viewing and converting images, and has very little functionality compared to the full version of the Elements software.
Cell Profiler

CellProfiler is free, open-source software for quantitative analysis of biological images. No prior experience in programming or computer vision is required.

Forums

image.sc Forum

This forum’s focus is software-oriented aspects of scientific imaging, particularly (but not limited to) image analysis, processing, acquisition, storage, and management of digital scientific images. The primary objective is to foster independent learning for everyone in the community.
Researchgate

Ask a technical question to get answers from experts, or start a scientific discussion with your peers.
RefractiveIndex.info

Database of literature-sourced refractive indices of common materials.

Manuals

Macro Coding in ImageJ

Textbook from the Bioimage Data Analysis Course at EMBL-CMCI. Download and extract .zip file for sample code and .pdf of textbook. Kota Miura. (2015). Macro Programming in ImageJ version 2.1.0. Zenodo. 10.5281/zenodo.30256
